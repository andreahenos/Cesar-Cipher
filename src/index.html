Mósy's Lamp es una aplicación web para cifrar y descrifrar rápidamente textos digitales de caracter personal (tales como diarios personales, cuadernos de viaje, libreta de reflexiones, etc). El método que se usa es el cifrado César, un tipo de cifrado por sustitución que reemplaza cada letra del texto original por otra que se encuentra un número fijo de posiciones (desplazamiento) más adelante en el mismo alfabeto.



Bienestar

Sé tú mismo y disfruta de los beneficios de escribir sobre tus pensamientos o sentimientos.

Seguridad

Tus textos son sólo suyos. Protegemos tu privacidad para que puedas escribir libremente.

Pro-planeta

Incentivamos el uso de diarios digitales para ahorra papel y aportar al cuidado del medio ambiente.

Recuerda

Queremos que puedas capturar momentos importantes para tí de forma honesta, pero además sencilla.

Paso 1

Paso 2

Paso 3

Paso 4

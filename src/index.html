Acerca de Be.you


Be.you es una aplicación web para cifrar y descrifrar rápidamente textos digitales de caracter personal (tales como diarios personales, cuadernos de viaje, libreta de reflexiones, etc). El método que se usa es el cifrado César, un tipo de cifrado por sustitución que reemplaza cada letra del texto original por otra que se encuentra un número fijo de posiciones (desplazamiento) más adelante en el mismo alfabeto.
